# 🏨 Hotel Management System

A comprehensive web-based platform for managing hotel operations, room bookings, and customer interactions, built with a robust Spring Boot backend and an elegant, user-friendly frontend.

## 🎓 Seminar Overview
This project demonstrates modern full-stack development principles using the Java ecosystem. It focuses on scalability, secure data handling, and professional UI/UX design.

## 🛠️ Tech Stack
- **Backend Core**: Java 8, Spring Boot 2.1.4, Spring MVC
- **Data Persistence**: Spring Data JPA with Hibernate
- **Database**: H2 (Memory-based) / MySQL
- **View Layer**: JSP (JavaServer Pages) with JSTL
- **Styling**: Vanilla CSS3 (Custom Premium Theme)
- **Security**: Form-based authentication and JSR-303 Validation
- **Build Tool**: Maven

## ✨ Key Features
- **User Management**: Robust registration and login system with real-time field validation.
- **Hotel Search**: City-wise search functionality to find the best accommodations.
- **Booking Engine**: Seamless reservation process for users.
- **Admin Dashboard**: Specialized interface for managing hotel listings and booking statuses.
- **Payment Integration**: Module for processing hotel stay payments.
- **Premium UI**: Modern "Pink & Beige" theme with responsive layouts and smooth transitions.

## 🚀 Getting Started
1. **Prerequisites**: Java 8 and Maven installed.
2. **Setup**: Clone the repository and navigate to the root directory.
3. **Execution**:
   ```powershell
   mvn spring-boot:run -DskipTests
   ```
4. **Access**:
   - Application: [http://localhost:8089](http://localhost:8089)
   - Database Console: [http://localhost:8089/h2-console](http://localhost:8089/h2-console)
